# Contributing to Egov.Integrations.MPass.Saml

Thank you for your interest in contributing to our project!

## How to Contribute

1.  **Report Bugs:** If you find a bug, please open an issue with a detailed description and steps to reproduce.
2.  **Suggest Features:** We welcome new ideas! Open an issue to discuss your proposal.
3.  **Submit Pull Requests:**
    *   Fork the repository.
    *   Create a new branch for your changes.
    *   Write tests for your changes.
    *   Ensure all tests pass.
    *   Submit a pull request with a clear description of your changes.

## Coding Standards

*   Follow the existing code style in the project.
*   Use meaningful variable and method names.
*   Add documentation/comments where necessary.
